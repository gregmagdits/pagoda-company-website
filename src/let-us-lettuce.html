// Let Us Lettuce

Find out what veggies are in season, as well as tasty recipes that use them.

WHAT IS THIS PROJECT ABOUT?

Before moving on to learning Angular, I wanted to challenge myself to produce a single page application from scratch. Let Us Lettuce was also a chance for me to put ES6 classes to use by modularizing all my services and controllers.

Upon page load, the Produce service requests a list of produce corresponding to the browser's local time. The produce API, built with node.js and Express, returns JSON data of whatever produce is in season for that month, and the UI controller iterates over the data and dynamically displays a card for each recipe, and handles displaying any errors. Users may then select any produce they like, and search for recipes that make use of it. The Recipe service initiates a call to the Edamam Recipe API, which returns recipes which contain that ingredient.

In the future, I would like to improve the app by optimizing the recipe search. I would like to see it return recipes that made the most optimal or efficient use of all the available produce, as opposed to just the selected ingredients. It would be cool to see a recipe come up that was the singular embodiment of that season.

Features

  • Updates veggies according to browser's localtime
  • Searches 1.5 million recipes via Edamam Recipe API
  • Single page application using vanilla JavaScript
  • Custom REST API back end

Built With

  • node.js
  • Express
  • MongoDB
  • Mongoose
  • Bootstrap
