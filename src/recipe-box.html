// Recipe Box

Collect and store your favorite recipes with Recipe Box.

WHAT IS THIS PROJECT ABOUT?

This was my first shot at making a full web application. Recipe Box allows users to create and edit their own private repository of recipes, complete with ingredients, instructions, and a featured image. The application is capable of registering, authenticating, and authorizing users, and the session data is maintained by passport.js. The pages are served by Express, which uses EJS templating. An interesting challenge for this project was how to handle recipe image uploads, as well as being able to update to a different image.

Features

  • Create and privately store your own recipes
  • Upload an image to feature in each recipe
  • User registration, authentication and authorization
  • Custom REST API back end
  • Server side templating with EJS

Built With

  • node.js
  • Express
  • MongoDB
  • Mongoose
  • EJS Templates
  • Bootstrap
